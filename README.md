# GreenAja

Monorepo project dengan tiga bagian utama:

```
greenaja/
├── mobile/     # Expo React Native App (wix/react-native-ui-lib)
├── admin/      # Admin Panel (coming soon)
└── backend/    # REST API Backend (coming soon)
```

## Getting Started

### Mobile
```bash
cd mobile
npm install
npx expo start
```
